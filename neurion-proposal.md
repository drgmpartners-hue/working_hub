## 선택된 방향

### 방향: D(업무 자동화) + A(콘텐츠 제작) + 재무계산기 우선 개발

**선택 이유**: 일상 업무에서 즉시 체감할 수 있는 효율 향상. 콘텐츠 제작과 업무 자동화가 결합되면 영업/마케팅 + 내부관리 양쪽 모두 생산성 극대화.

### 예상 핵심 기능 (초안, 3개)

| # | 기능 | 설명 | 관련 아이디어 |
|---|------|------|-------------|
| 1 | Working Hub 대시보드 | 모든 프로그램에 접근하는 중앙 허브 UI | 전체 |
| 2 | 콘텐츠 제작 스튜디오 | 카드뉴스/PPT/홍보페이지/보고서표지를 하나의 인터페이스에서 제작 | #1, #2, #4, #5, #11 |
| 3 | 업무 자동화 엔진 | 한글/워드 문서 자동화, 회계관리, CRM, 경영지표 대시보드 통합 | #3, #6, #12, #15, #18, #22 |

### 예상 타겟 사용자

- **직접 사용자**: 종합자산관리 회사 직원 (영업, 마케팅, 관리 부서)
- **간접 수혜자**: 병의원 원장 고객 (제작된 콘텐츠/보고서의 수신자)
