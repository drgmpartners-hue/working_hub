## 화면 상세

### SCR-01: 로그인
- **ID**: SCR-01
- **경로**: /login
- **기능**: 사용자 인증
- **컴포넌트**:
  - 로고 이미지
  - 이메일 입력 필드
  - 비밀번호 입력 필드
  - 로그인 버튼
  - 에러 메시지 영역
- **진입**: 앱 최초 접근, 세션 만료
- **이탈**: 로그인 성공 → 대시보드

### SCR-02: 대시보드
- **ID**: SCR-02
- **경로**: /dashboard
- **기능**: 모든 프로그램 접근 런처/허브
- **레이아웃**: 카테고리별 아코디언 + 카드 그리드 (화면 중앙)
- **컴포넌트**:
  - 공통 헤더 바
  - 카테고리 그룹 (업무 자동화 / 투자 분석 / 콘텐츠 제작)
  - 카테고리 클릭 시 하위 프로그램 카드 펼침
  - 프로그램 카드 (아이콘, 이름, 설명)
- **진입**: 로그인 성공, 헤더 홈 버튼
- **이탈**: 프로그램 카드 클릭 → 해당 프로그램 페이지 (새 페이지)

### SCR-03: Dr.GM 수당정산 계산기
- **ID**: SCR-03
- **경로**: /commission/dr-gm
- **기능**: 엑셀 업로드 → 수당 계산 → 명세서 다운로드
- **탭 구조**: 2탭
- **컴포넌트**:
  - 공통 헤더 바
  - **탭 1 (업로드 & 계산)**:
    - 엑셀 파일 업로드 영역 (상단)
    - 계산 결과 데이터 테이블 (하단)
    - "다음" 버튼
  - **탭 2 (미리보기 & 다운로드)**:
    - 개인별 수당명세서 미리보기
    - 다운로드 버튼
    - "이전" 버튼
- **진입**: 대시보드 → 업무 자동화 → Dr.GM 카드 클릭
- **이탈**: 헤더 홈 → 대시보드

### SCR-04: 증권사 수당정산 계산기
- **ID**: SCR-04
- **경로**: /commission/securities
- **기능**: 웹 크롤링/엑셀 → 템플릿 적용/계산 → 보고서 다운로드
- **탭 구조**: 3탭
- **컴포넌트**:
  - 공통 헤더 바
  - **탭 1 (데이터 확인)**:
    - 크롤링 실행 버튼 or 엑셀 업로드 영역
    - 데이터 결과 확인 테이블
    - "다음" 버튼
  - **탭 2 (템플릿 & 계산)**:
    - 템플릿에 적용된 결과 표시
    - 계산 수행 / 결과 표시
    - "이전" / "다음" 버튼
  - **탭 3 (미리보기 & 다운로드)**:
    - 보고서 미리보기
    - 다운로드 버튼
    - "이전" 버튼
- **진입**: 대시보드 → 업무 자동화 → 증권사 수당정산 카드 클릭
- **이탈**: 헤더 홈 → 대시보드

### SCR-05: IRP 포트폴리오 수익률 관리기
- **ID**: SCR-05
- **경로**: /portfolio/irp
- **기능**: 데이터 수집 → 분석/편집/AI → 보고서 PDF
- **탭 구조**: 3탭
- **컴포넌트**:
  - 공통 헤더 바
  - **탭 1 (데이터 확인)**:
    - 크롤링 실행 버튼 or 엑셀 업로드 영역
    - 데이터 결과 확인 테이블
    - "다음" 버튼
  - **탭 2 (템플릿 & 편집 & AI 분석)**:
    - 템플릿에 적용된 결과 표시
    - 직접 값 입력/수정 가능 영역 (리밸런싱, 추천상품)
    - AI 분석 결과 표시 영역
    - "이전" / "다음" 버튼
  - **탭 3 (보고서 & PDF)**:
    - 보고서 미리보기
    - PDF 다운로드 버튼
    - "이전" 버튼
- **진입**: 대시보드 → 투자 분석 → IRP 포트폴리오 카드 클릭
- **이탈**: 헤더 홈 → 대시보드

### SCR-06: 주식/ETF 추천 프로그램
- **ID**: SCR-06
- **경로**: /investment/stock-recommend
- **기능**: 테마 분석/선택 → 종목 확인/포트폴리오 담기
- **단계 구조**: 2단계 (위저드)
- **컴포넌트**:
  - 공통 헤더 바
  - **Step 1 (테마 분석 & 선택)**:
    - 테마 목록 (증권사 API)
    - AI 분석 점수 표시
    - 추천 테마 하이라이트
    - 테마 바구니 담기 (복수 선택)
    - "다음" 버튼
  - **Step 2 (종목 확인 & 포트폴리오)**:
    - 선택 테마별 주식 목록
    - Top 5 구별 표시
    - 주식 상세 정보 (기간별 수익률, 기관 매수, 외국인 매수)
    - "내 포트폴리오에 담기" / "회사풀에 담기" 버튼
    - "이전" 버튼
  - **팝업: 주식 분석 보고서**:
    - 개별 주식 클릭 시 팝업
    - NotebookLM식 리서치 기반 AI 분석 보고서
    - 닫기 버튼
- **진입**: 대시보드 → 투자 분석 → 주식/ETF 추천 카드 클릭
- **이탈**: 헤더 홈 → 대시보드

### SCR-07: 카드뉴스 제작
- **ID**: SCR-07
- **경로**: /content/card-news
- **기능**: 주제 입력 → AI 작성 → 브랜드 적용 생성 → 편집 → 다운로드
- **컴포넌트**:
  - 공통 헤더 바
  - 주제/내용 입력 영역
  - AI 글 콘텐츠 표시/검토 영역
  - 승인/수정 버튼
  - 카드뉴스 미리보기 (브랜드 적용)
  - 텍스트 수정 → 재생성 기능
  - 이미지 부분 수정 요청 → 재생성 기능
  - 다운로드 버튼
- **진입**: 대시보드 → 콘텐츠 제작 → 카드뉴스 카드 클릭
- **이탈**: 헤더 홈 → 대시보드
