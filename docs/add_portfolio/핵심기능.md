그럼 irp/연금저축 포트폴리오 관리 페이지의 수정사항을 본격적으로 진행해보자.

프로그램에서는 1~3단계의 내용은 연결해서도 할 수 있어야 하지만 각각 진행도 가능해야해.

1단계는
- 고객정보를 입력하는 기능. (기존 고객 선택 or 신규 입력, 여러 명 배치)
- 불러온 고객정보에 증권사캡쳐 화면이랑 매칭시키는 기능. (Ctrl+V 붙여넣기 핵심!)

2단계는
- 고객정보를 불러오는 기능.
- 불러온 정보 중 보고서로 만들 정보를 선택하는 기능.
- 선택한 정보를 바탕으로 포트폴리오표, 수익률 그래프(3개월, 6개월, 1년)를 확인하는 기능.
- 추가: 지역 분산 그래프, 위험도 분산 그래프

3단계는
- 이렇게 선택한 정보를 바탕으로 보고서를 작성하는 기능.
  - 개요
  - 포트폴리오 분석: 수익률 표 + 지역분산 그래프 + 위험도 분산 그래프 + 기간별 수익률 + AI 코멘트
  - 포트폴리오 변경 안내 + AI 코멘트
- 작성된 보고서를 pdf로 다운받는 기능.
