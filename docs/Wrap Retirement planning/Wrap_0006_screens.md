## 공통 요소

### COMMON-01: 고객 선택 바
- **ID**: common-customer-selector
- **위치**: 탭 아래, 콘텐츠 상단 고정
- **컴포넌트**:
  - 고객 검색/선택 드롭다운

### COMMON-02: 탭 네비게이션
