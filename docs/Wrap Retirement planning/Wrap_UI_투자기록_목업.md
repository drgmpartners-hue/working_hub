## 3. 투자기록 추가 모달

```
┌─────────────────────────────────────────────┐
│           투자기록 추가                       │
├─────────────────────────────────────────────┤
│                                              │
│  유형:  ○ 신규투자  ○ 추가적립  ○ 인출      │
│                                              │
│  상품명:    [KB증권 랩B-01          ]        │
│  투자금액:  [          6,200  ] 만원         │
│  시작일:    [  2025-03-25     ]              │
│  상태:      ○ 운용중(ing)  ○ 종결(exit)     │
│                                              │
│  ── exit 시 추가 입력 ──                     │
│  종료일:    [  2025-06-15    ]               │
│  평가금액:  [         6,572  ] 만원          │
│  수익률:    자동계산 → +6.0%                 │
│                                              │
│  ── 연결 정보 ──                             │
│  선행상품:  [#1 랩A-01 (exit)     ▼]         │
│  후행상품:  [미지정                ▼]         │
│                                              │
│  메모: [                                ]    │
│                                              │
│         [취소]              [저장]            │
└─────────────────────────────────────────────┘
```

## 4. 핵심 인터랙션

| 동작 | 설명 |
|------|------|
| **연결상품 클릭** | 해당 상품 행으로 스크롤 + 하이라이트 |
| **상태 토글** | ing → exit 변경 시 종료일/평가금액 입력 팝업 |
| **타임라인 뷰 토글** | 같은 데이터를 시간축 시각화로 전환 |
| **연간 흐름표 자동계산** | 투자기록 입력 시 해당 연도 흐름표 자동 업데이트 |
| **필터** | 전체/운용중/종결/적립 필터링 |
