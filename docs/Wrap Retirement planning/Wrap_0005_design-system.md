## 3. 레이아웃

### 전체 구조
```
┌─────────────────────────────────────────────────────────┐
│  Hub Header (공통)                                       │
├─────────────────────────────────────────────────────────┤
│  Hub Sidebar (공통)                                      │
│  ┌───────────────────────────────────────────────────┐  │
│  │  고객 선택 바 (상단 고정)                           │  │
│  │  [고객 선택 ▼]  홍길동(C-001) | 목표: 10억 | 은퇴: 65세 │
│  ├───────────────────────────────────────────────────┤  │
│  │  [1번탭] [2번탭] [3번탭] [4번탭] [5번탭]            │  │
│  ├───────────────────────────────────────────────────┤  │
│  │                                                    │  │
│  │              탭 콘텐츠 영역                          │  │
│  │              (각 탭별 내용)                          │  │
│  │                                                    │  │
│  └───────────────────────────────────────────────────┘  │
└─────────────────────────────────────────────────────────┘
```

### 고객 선택 바
- 높이: 56px
- 배경: Surface (#F5F7FA)
- 고객 드롭다운: 좌측 정렬
- 기본정보: 고객명(고유번호) | 목표은퇴자금 | 희망은퇴나이
- 하단 보더: Border (#E1E5EB) 1px

### 탭 네비게이션
- 스타일: 밑줄 탭 (underline tab)
- 활성 탭: Primary color + 2px 하단 보더
- 비활성 탭: Text Secondary color
- 탭 높이: 48px

## 4. 핵심 컴포넌트

### 금융 테이블
- 숫자 우측 정렬 (Inter 서체, 고정폭)
- 천단위 콤마 자동 적용
- 음수: Error Red (#EF4444)
- 양수 수익률: Success Green (#10B981)
- 음수 수익률: Error Red (#EF4444)
- 헤더: Surface 배경, Text Primary, font-weight 600
- 행 호버: #F5F7FA
- 행 높이: 48px

### 성장 그래프 (1, 3, 4, 5번탭 공통)
- 차트 타입: Area Chart + Line Chart 결합
- X축: 연도 (또는 나이)
- Y축: 금액 (만원, 천단위 콤마)
- 그리드: 수평선만, #E1E5EB
- 범례: 차트 상단 우측
- 반응형: 컨테이너 width 100%

### 투자기록 상태 배지
```
🔵 운용중  → 배경: #EFF6FF, 텍스트: #3B82F6, 보더: #3B82F6
🟢 종결    → 배경: #ECFDF5, 텍스트: #10B981, 보더: #10B981
💰 입금    → 배경: #FEF9EC, 텍스트: #D4A847, 보더: #D4A847
```

### 괴리율 표시 (4번탭)
```
목표 상회: +5.2%  → Success Green, 상향 화살표 ↑
목표 부합: ±2%    → Info Blue, 수평선 →
목표 하회: -8.3%  → Error Red, 하향 화살표 ↓, AI 가이드 버튼 활성화
```

## 5. 반응형

| 브레이크포인트 | 레이아웃 |
|-------------|---------|
| Desktop (> 1280px) | 풀 레이아웃, 테이블 전체 컬럼 표시 |
| Laptop (1024-1280px) | 테이블 일부 컬럼 숨김 (hover 시 표시) |
| Tablet (768-1024px) | 탭 스크롤, 테이블 횡스크롤 |
| Mobile (< 768px) | 지원하지 않음 (업무용 데스크톱 도구) |
